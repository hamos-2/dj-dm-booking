# 🖥️ Admin Dashboard 화면 정의서

> `app/admin/` 기준  
> Next.js App Router + 단일 어드민 사용자 가정

---

## 전체 화면 구조

```
Admin Dashboard
├── [A] 대시보드 홈          /admin
├── [B] 예약 목록            /admin/bookings
├── [C] 예약 상세            /admin/bookings/[id]
├── [D] Instagram DM 인박스  /admin/instagram
└── [E] 설정
    ├── 가용 시간 설정        /admin/settings/availability
    └── Google 연동          /admin/settings
```

## 공통 레이아웃 (Admin Shell)

**파일**: `app/admin/layout.tsx`

```
┌─────────────────────────────────────────────────┐
│  ☰  BookingAdmin                    [로그아웃]   │  ← 헤더 (h-14)
├──────────────┬──────────────────────────────────┤
│              │                                  │
│  📊 대시보드  │                                  │
│  📋 예약 목록 │         (메인 콘텐츠)             │
│  💬 DM 인박스 │                                  │
│     [●] 3   │                                  │  ← DM 뱃지: 미확인 수
│  ⚙️  설정    │                                  │
│              │                                  │
└──────────────┴──────────────────────────────────┘
   사이드바(w-56)         메인 영역
```

**컴포넌트 요소:**
- 사이드바: 현재 경로 active 표시
- DM 인박스 메뉴에 미확인 DM 뱃지 (숫자)
- 헤더: 현재 페이지 타이틀 표시
- Auth Guard: 비로그인 시 `/login`으로 redirect

## [A] 대시보드 홈 `/admin`

**목적:** 오늘 일정 한눈에 파악

```
┌─────────────────────────────────────────────────┐
│  오늘의 예약            2025년 6월 10일 (화)      │
├─────────────────────────────────────────────────┤
│                                                 │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐        │
│  │  오늘    │  │ 이번 주  │  │ 미확인DM │        │
│  │  3건    │  │  12건   │  │   2건   │        │
│  └─────────┘  └─────────┘  └─────────┘        │
│                                                 │
│  오늘 예약 타임라인                              │
│  ─────────────────────────────────────         │
│  09:00  ●  홍길동  (웹)            [상세보기]   │
│  11:15  ●  김철수  (Instagram)     [상세보기]   │
│  14:00  ○  (비어있음)                          │
│  15:15  ●  이영희  (웹)            [상세보기]   │
│                                                 │
└─────────────────────────────────────────────────┘
```

**데이터:**
- 오늘 날짜의 confirmed 예약 목록 (시간순 정렬)
- 각 예약: 시간, 고객명, 소스(웹/Instagram), [상세보기] 버튼
- 카드: 오늘 예약 수 / 이번 주 예약 수 / 미확인 DM 수

```
┌─────────────────────────────────────────────────┐

│                                                 │
│  [전체▼]  [날짜 범위: 06/01 ~ 06/30]  [검색 🔍] │  ← 필터 영역
│                                                 │
│  ┌────┬────────┬──────────┬──────┬───────┬───┐ │
│  │날짜│  시간   │  고객명  │ 소스  │ 상태  │   │ │
│  ├────┼────────┼──────────┼──────┼───────┼───┤ │
│  │6/10│09:00   │ 홍길동   │ 웹   │ ✅확정 │ ⋮ │ │
│  │6/10│11:15   │ 김철수   │ DM   │ ✅확정 │ ⋮ │ │
│  │6/09│14:00   │ 이영희   │ 웹   │ ❌취소 │ ⋮ │ │
│  └────┴────────┴──────────┴──────┴───────┴───┘ │

```

**필터 옵션:**
- 상태: 전체 / 확정 / 취소
- 날짜 범위: date range picker
- 검색: 고객명 / 연락처

**행 액션 (⋮ 메뉴):**
- 상세보기 → `/admin/bookings/[id]`
- 리스케줄
- 취소

### [C-1] 리스케줄 모달

**트리거:** [리스케줄] 버튼 클릭

```
┌──────────────────────────────┐
│  리스케줄                 ✕  │
│                              │
│  현재: 06/10 09:00~10:00     │
│                              │
│  새 날짜  [06 / 12 ▼]        │
│                              │
│  가용 시간                   │
│  ┌──────┐ ┌──────┐ ┌──────┐ │
│  │09:00 │ │10:15 │ │11:30 │ │
│  └──────┘ └──────┘ └──────┘ │
│  ┌──────┐ ┌──────┐           │
│  │14:00 │ │15:15 │           │
│  └──────┘ └──────┘           │
│                              │
│  선택됨: 14:00~15:00          │
│                              │
│  [취소]          [변경 확정]  │
└──────────────────────────────┘
```

**동작:**
1. 날짜 선택 → getAvailableSlots 호출 (excludeBookingId 포함)
2. 슬롯 클릭 → 선택 상태 표시
3. [변경 확정] → rescheduleBooking 호출
4. 성공 시 모달 닫기 + 예약 상세 업데이트 + Toast 알림

## 상태 뱃지 정의

| 상태 | 색상 | 텍스트 |
|------|------|--------|
| confirmed | 🟢 Green | 확정 |
| canceled | 🔴 Red | 취소 |
| (DM) 미확인 | 🔵 Blue dot | 뱃지로 표시 |

---

## API 연결 요약

| 화면 | 호출 API |
|------|----------|
| 대시보드 홈 | DB query (오늘 예약) |
| 예약 목록 | DB query (필터) |
| 예약 상세 | DB query (단건) |
| 리스케줄 모달 | getAvailableSlots + rescheduleBooking |
| 취소 모달 | cancelBooking |
| DM 인박스 | DB query (instagram_messages) |
| DM 예약 확정 모달 | getAvailableSlots + createBooking |
| 가용시간 설정 | DB upsert (availability) |
| Google 연동 | OAuth 2.0 flow |
